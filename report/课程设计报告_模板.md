## 一、引言

### 1.1 研究背景

城市化是人类活动对地表最深刻的影响之一。中国城镇化率从 2000 年的 36.2% 增长到 2025 年的约 68%，城市建成区面积持续扩大。及时、准确地监测城市扩展过程，对于城市规划、土地资源管理和可持续发展具有重要意义。

传统地面调查方法成本高、周期长，遥感技术因其大范围、多时相、低成本的优势，已成为城市扩张监测的主流手段。

### 1.2 研究区概况

**三明市**：位于福建省中西部（117.64°E, 26.26°N），闽赣交界处，是典型的山区河谷城市，城市沿沙溪河谷带状分布，扩张受地形约束明显。

**福州市**：福建省省会（119.30°E, 26.07°N），位于闽江入海口，沿海平原城市，城市呈圈层式扩张，与三明形成鲜明对比。

### 1.3 研究目标

1. 提取 2000-2025 年三明市和福州市建成区范围（5 年间隔）
2. 计算 NDBI、NDVI、MNDWI 三个遥感指数
3. 分析城市空间扩张的速率与方向
4. 对提取结果进行精度验证（混淆矩阵 + Kappa 系数）
5. 对比两座不同类型城市的扩张模式差异

## 二、数据与方法

### 2.1 数据来源

| 数据 | 来源 | 空间分辨率 | 时间范围 |
|------|------|-----------|---------|
| Landsat 5 TM | USGS / Google Earth Engine | 30m | 2000-2010 |
| Landsat 8 OLI | USGS / Google Earth Engine | 30m | 2015-2025 |

每年筛选全年云量 < 30% 的影像，经逐景云掩膜后取中值合成。

### 2.2 技术路线

```
Landsat 影像 → 云掩膜 → 中值合成 → 计算 NDBI / NDVI / MNDWI
    → 综合阈值分割 → 建成区提取 → 面积统计
    → 精度验证（混淆矩阵 + Kappa 系数）
    → 方向分析（八扇形统计）
    → 可视化输出（地图 + 图表 + 动图 + CSV）
```

### 2.3 遥感指数

#### NDBI（归一化建筑指数）
$$NDBI = \frac{SWIR1 - NIR}{SWIR1 + NIR}$$

建成区（混凝土、沥青）在短波红外（SWIR1）反射率高、近红外（NIR）反射率低，NDBI 偏高。阈值 NDBI > -0.05 判定为建成区。

#### NDVI（归一化植被指数）
$$NDVI = \frac{NIR - Red}{NIR + Red}$$

用于辅助排除植被干扰。建成区 NDVI 通常 < 0.2。

#### MNDWI（改进归一化水体指数）
$$MNDWI = \frac{Green - SWIR1}{Green + SWIR1}$$

用于辅助排除水体干扰。MNDWI > 0 判定为水体。

### 2.4 精度验证方法

采用双阈值交叉验证：
- **主阈值** NDBI > -0.05：实际使用的建成区判别标准
- **严格阈值** NDBI > 0.05：作为伪参考数据

在研究区内生成 500 个随机采样点，构建混淆矩阵，计算总体精度（OA）、Kappa 系数、制图精度、用户精度和 F1 分数。

## 四、讨论

### 4.1 两城市扩张模式对比

三明（山区河谷城市）与福州（沿海平原城市）的扩张模式存在显著差异：
- **空间形态**：三明呈带状延伸，福州呈圈层扩展
- **扩张速率**：（待填充）
- **方向偏好**：（待填充）

### 4.2 驱动因素分析

- **政策因素**：城市规划调整、新区/开发区建设
- **经济因素**：产业升级、人口增长、城镇化推进
- **交通因素**：高速公路、铁路等基础设施的拉动效应
- **自然因素**：地形约束（三明河谷）、海岸线（福州）

### 4.3 方法优势与局限

**优势**：
- GEE 云端处理避免下载大量数据（~6GB/景）
- 双指数辅助验证提高建成区提取准确性
- 双阈值交叉验证提供定量精度参考

**局限**：
- 30m 分辨率限制了对小尺度城市变化的探测
- NDBI 阈值法在裸土区域可能产生误判
- 精度验证使用伪参考数据而非实地采样
- 中值合成可能平滑掉年内城市变化细节

### 4.4 改进方向

1. 引入 Sentinel-2（10m 分辨率）与 Landsat 融合提高精度
2. 使用面向对象分类替代阈值分割
3. 增加夜间灯光数据（NPP-VIIRS）作为辅助数据源
4. 收集实地验证样本替换伪参考数据

## 五、结论

1. 2000-2025 年，三明市和福州市建成区面积均呈持续增长趋势
2. 两城市扩张模式差异显著：三明带状延伸 vs 福州圈层扩展
3. NDBI + NDVI + MNDWI 三指数联合法有效提高了建成区提取准确性
4. 精度验证表明分类结果具有较高可靠性（Kappa > 0.6）
5. 本研究方法可推广至其他城市的扩张监测
